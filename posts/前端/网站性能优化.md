## 字体切片

中文字体动辄几 MB，首屏只用几十个字却要全量下载。字体切片就是把字体按“需要的字符”拆分，让浏览器按需加载。

### 有什么效果

- 字体文件体积显著下降，首屏更快。  
- 加载更可控，不常用字符不会拖慢页面。  
- 对中文站点和内容型网站提升明显。  

### 常用方式

1. 按字符白名单：只保留页面会出现的字符。  
2. 按区间切片：按 `unicode-range` 拆分，按需加载。  
3. 配合缓存：切片稳定，命中率更高。  

### 怎么用（font-slice）

工具地址： https://github.com/voderl/font-slice

安装：
```bash
npm install --save-dev font-slice
# 或
yarn add -D font-slice
```

使用：
```js
const createFontSlice = require('font-slice');
const path = require('path');

createFontSlice({
  fontPath: path.resolve(__dirname, 'YourPath.ttf'),
  outputDir: path.resolve(__dirname, './output'),
});
```
