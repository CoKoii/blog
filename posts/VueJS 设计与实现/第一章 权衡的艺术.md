## 命令式和声明式

<a href="https://chat.baidu.com/search?word=%E8%A7%86%E5%9B%BE%E5%B1%82%E6%A1%86%E6%9E%B6%E6%98%AF%E4%BB%80%E4%B9%88" target="_blank" rel="noopener">视图层框架</a>
通常分为 `命令式` 和 `声明式`,它们各有优缺点

在设计框架时,应该对两种范式都有足够的认知,这样才能做出正确的选择,将两者的优点捏合🤌

### `命令式`

命令式框架的一大特点就是**关注过程**

例如,我们把下面这段话翻译成对应的代码:

```js
- 获取 id 为 app 的 div 标签
- 它的文本内容为 hello world
- 为其绑定点击事件
- 当点击时弹出提示: ok
```

对应的代码为:

```js
const div = document.querySelector('#app')
div.innerText = 'hello world'
div.addEventListener('click', () => {
  alert('ok')
})
```
